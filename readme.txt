Pattern post manager is a plugin to make Writing more efficient.
Registering a pattern saves you the trouble of entering the same wording.

== Description ==

It is a plug-in that inserts registered template text into a visual editor such as posting, fixed page, custom posting.

The registered formal sentence is set as a list box in the toolbar of posting or fixed page visual editor, and the selected fixed form sentence is inserted in the editor.

**After inserting into an article, even if you change the original fixed form, the article will not change.**

= Supported languages =
* English(us)
* Japanese(ja)


== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "pattern post manager"
3. **Activate** Pattern post manager from your Plugins page


== Frequently asked questions ==
= After inserting a fixed form sentence into an article, if you change the template, will the article change too? =
It will not be changed.<br>Just inserting fixed form sentences into articles, we do not link fixed form and articles.
